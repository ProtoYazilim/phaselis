---
title: Linear Gradients Interface
sidebar_position: 2
description: Inspect the linear gradient interface.
---

linear gradient attributes can be passed via style prop to the `Block` component. 

pass the linear gradient props to the `Block` component to apply the gradient to the background of the block.

```tsx
<Block style={{ backgroundColor: "linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1))" }}>
  <Text>Hello</Text>
</Block>
```
