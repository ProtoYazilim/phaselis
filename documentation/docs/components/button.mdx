The `Button` component is a customizable button for React Native that supports various styles and icons. It works in integration with context providers to simplify form handling.

## Import

Phaselis exports one main component for Button.

```js
import { Button } from 'phaselis'
```

- **Button:** Main Component import for Button

## Usage

### Basic Button

Put in some content in the `Button` to get a basic button.

```jsx
import { Button } from "phaselis";

function BasicUsage(): React.ReactElement {
  return <Button text="Phaselis" primary />;
}

export default BasicUsage; 
```

### disabled 
`boolean`

If `true`, the button will be disabled.

```jsx
<Button disabled />
```



### onClick 
`((event: GestureResponderEvent) => void) | null | undefined`

Function to execute when the button is pressed.

```jsx
<Button onClick={() => console.log("Button clicked")} />
```

### type 
`submit | reset | button`

If compenent is used in a form, you can set the type of the button.

```jsx
<Button type="submit" /> 
```

### style
| Name | Type |
| ---- | ---- |
| `container` | ViewStyle |
| `text` | TextStyle |
| `leftSlot` | ViewStyle |
| `rightSlot` | ViewStyle |

You can style the button with the `style` prop using React.CSSProperties.

```jsx  
<Button
    style={{
      container: {
        backgroundColor: "red",
      },
      text: {
        color: "white",
      },
    }}
  />
```

### loading
`boolean`

If `true`, the button will show a loading spinner.

```jsx
<Button loading />  
```

### size 
`xs | sm | md | lg | xl | xxl`

The size of the button.

```jsx
<Button size="xl" />  
```

### primary
`boolean`

If `true`, the button will have a primary color.

```jsx
<Button primary />  
```

### secondary
`boolean`

If `true`, the button will have a secondary color.

```jsx
<Button secondary />  
```

### tertiary
`boolean`

If `true`, the button will have a tertiary color.

```jsx
<Button tertiary />  
```

### outline
`boolean`

If `true`, the button will have an outline.

```jsx
<Button outline />  
```

### LeftSlot
`ReactNode`

You can pass any component to the left slot of the button.

```jsx
import { LucideIcon } from "phaselis";

{...}
  <Button LeftSlot={() => <LucideIcon name={"ArrowLeft"} />} />;
{...}
```

### RightSlot
`ReactNode`

You can pass any component to the right slot of the button.

```jsx
import { LucideIcon } from "phaselis";

{...}
  <Button RightSlot={() => <LucideIcon name={"ArrowRight"} />} />;
{...}
```

### leftIcon
`Lucide icon name`

You can pass any Lucide icon name to the left slot of the button.

```jsx
<Button leftIcon="ArrowLeft" />
```

### rightIcon
`Lucide icon name`

You can pass any Lucide icon name to the right slot of the button.

```jsx
<Button rightIcon="ArrowRight" />;
```

### children
`ReactNode`

You can pass any component to the button.

```jsx
import { Text } from "react-native";

{...}
  <Button>
    <Text>Click me</Text>
  </Button>
{...}
```
