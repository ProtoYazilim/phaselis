# TabViewHeader

The `TabViewHeader` component is used to customize and control the appearance and behavior of the tab headers in a `TabView`.

## Usage

```jsx
<TabView
  headerProps={{
    data: [{ text: 'Overview' }, { text: 'Details' }, { text: 'More' }],
    activeTab: 1,
    handleOnPress: (index) => console.log(`Tab ${index} pressed`),
    showIcons: true,
    scrollable: true,
    style: { backgroundColor: "#e0e0e0", padding: 5 },
  }}
>
  <TabItem title="Overview">
    <Text>Overview Content</Text>
  </TabItem>
  <TabItem title="Details">
    <Text>Details Content</Text>
  </TabItem>
  <TabItem title="More">
    <Text>More Content</Text>
  </TabItem>
</TabView>
```

## Props

### data
`any[]`

The data array used to display tab headers, typically derived from the `title` props of `TabItem` components.


```jsx
<TabView
  headerProps={{
    data: [{ text: 'Tab 1' }, { text: 'Tab 2' }],
  }}
>
  {/* Tabs go here */}
</TabView>
```

### activeTab
`number`

The index of the currently active tab.


```jsx
<TabView
  headerProps={{
    activeTab: 0,
  }}
>
  {/* Tabs go here */}
</TabView>
```

### handleOnPress
`(index: number) => void`

Function called when a tab header is pressed. It is used to set the active tab and handle any additional behavior needed on tab click.


```jsx
<TabView
  headerProps={{
    handleOnPress: (index) => console.log(`Tab ${index} pressed`),
  }}
>
  {/* Tabs go here */}
</TabView>
```

### style
| Name        | Type      | Description                                      |
|-------------|-----------|-------------------------------------------------- |
| `container` | ViewStyle | Style for the outer container of the tab header.   |
| `element`   | ViewStyle | Style for the individual tab elements within the header.    |

```jsx
<TabView
  headerProps={{
    style: {
      container: { backgroundColor: "#e0e0e0", padding: 10 },
      element: { borderBottomWidth: 2, borderColor: "blue" },
    },
  }}
>
  <TabItem title="Home">
    <Text>Home Content</Text>
  </TabItem>
  <TabItem title="Settings">
    <Text>Settings Content</Text>
  </TabItem>
</TabView>
```

### scrollable
`boolean`

Indicates if the tab headers should be scrollable when they exceed the container width.


```jsx
<TabView
  headerProps={{
    scrollable: true,
  }}
>
  {/* Tabs go here */}
</TabView>
```

### leftIcon / rightIcon
`SlotIconName`

Optional icons that can be displayed on the left or right side of the tab headers.


```jsx
<TabView
  headerProps={{
    leftIcon: "ArrowLeft",
    rightIcon: "ArrowRight",
  }}
>
  {/* Tabs go here */}
</TabView>
```
