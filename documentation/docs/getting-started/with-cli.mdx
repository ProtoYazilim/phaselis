# Quick Start With CLI

## Prerequisites

- Before we start our project follow along expo [Set up your environment](https://docs.expo.dev/get-started/set-up-your-environment/?platform=ios&device=simulated&mode=development-build&buildEnv=local) section or [react-native without framework guide](https://reactnative.dev/docs/getting-started-without-a-framework) to prepare your local development environment.

## Installation

- Open a Terminal in your project's folder

<Tabs>
  <TabItem value="npm" label="npm" default>
  ```bash npm2yarn
  npx create-phaselis-app
  ```
  </TabItem>
  <TabItem value="yarn" label="yarn">
  ```bash npm2yarn
  yarn create phaselis-app
  ```
  </TabItem>
</Tabs>

You would be prompted to choose a template for your project. You can choose between the following templates:

- **[Expo Template](https://github.com/expo/expo/tree/main/templates/expo-template-blank-typescript)**
- **[React Native Community Template](https://github.com/react-native-community/template/tree/main/template)**

After the installation is complete, you can navigate to your project folder by running the following command:

```bash
cd [project-name]
```

## Running the project

To run the project, you can use the following commands:

<Tabs>
  <TabItem value="npm" label="npm" default>
  ```bash npm2yarn
  npm start
  ```
  </TabItem>
  <TabItem value="yarn" label="yarn">
  ```bash npm2yarn
  yarn start
  ```
  </TabItem>
</Tabs>
* this will start the metro bundler.

to run the project on a specific platform you can use the following commands:

### IOS

<Tabs>
  <TabItem value="npm" label="npm" default>
  ```bash npm2yarn
  npm run ios
  ```
  </TabItem>
  <TabItem value="yarn" label="yarn">
  ```bash npm2yarn
  yarn ios
  ```
  </TabItem>
</Tabs>

### Android

<Tabs>
  <TabItem value="npm" label="npm" default>
  ```bash npm2yarn
  npm run android
  ```
  </TabItem>
  <TabItem value="yarn" label="yarn">
  ```bash npm2yarn
  yarn android
  ```
  </TabItem>
</Tabs>
