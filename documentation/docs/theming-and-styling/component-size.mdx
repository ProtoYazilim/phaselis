### Component Size Type

#### Description  
The `ComponentSize` type defines the possible size values for components, with corresponding pixel dimensions for consistent sizing throughout the framework.  

| Value  | Description          | 
|--------|----------------------|
| `xs` | Extra small size.    | 
| `sm` | Small size.          | 
| `md` | Medium size.         | 
| `lg` | Large size.          | 
| `xl` | Extra large size.    | 
| `xxl` | Extra extra large size. |
